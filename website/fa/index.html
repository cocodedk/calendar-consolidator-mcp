Calendar Consolidator MCP English

یکپارچه‌سازی تقویم‌های شما

تقویم‌های Microsoft 365 و iCloud/CalDAV را در یک تقویم متحد ادغام کنید با همگام‌سازی تدریجی، آینه‌کاری یک‌طرفه، و کنترل هوش مصنوعی از طریق پروتکل MCP.

Node.js Python پروتکل MCP SQLite متن‌باز
مشاهده در GitHub

ویژگی‌ها

  • پشتیبانی چندمنبعی: Microsoft Graph، CalDAV
  • همگام‌سازی یک‌طرفه: آینه‌کاری از منبع به تقویم هدف
  • به‌روزرسانی تدریجی: همگام‌سازی دلتا با تشخیص تغییرات
  • یکپارچه‌سازی MCP: کنترل هوش مصنوعی/عامل از طریق پروتکل MCP
  • رابط وب: پیکربندی ساده از طریق مرورگر
  • اولویت محلی: پایگاه داده SQLite، بدون وابستگی ابری

شروع سریع

  1. نصب Node.js 20+ و Python 3.12+
  2. کلون: git clone https://github.com/cocodedk/calendar-consolidator-mcp
  3. نصب وابستگی‌ها: npm install && pip install -r requirements.txt
  4. راه‌اندازی پایگاه داده: python python/init_db.py
  5. اجرا: npm start — سپس http://localhost:3000 را باز کنید

Unify Your Calendars

Consolidate Microsoft 365 and iCloud/CalDAV calendars into one unified calendar with one-way mirroring, incremental sync, and AI/agent control via MCP protocol.

Node.js Python MCP Protocol SQLite Open Source
View on GitHub

Features

  • Multi-source support: Microsoft Graph, CalDAV
  • One-way sync: Source to target calendar mirroring
  • Incremental updates: Delta sync with change detection
  • MCP integration: AI/agent control via MCP protocol
  • Web UI: Simple browser-based configuration
  • Local-first: SQLite database, no cloud dependencies

Quick Start

  1. Install Node.js 20+ and Python 3.12+
  2. Clone: git clone https://github.com/cocodedk/calendar-consolidator-mcp
  3. Install dependencies: npm install && pip install -r requirements.txt
  4. Initialize database: python python/init_db.py
  5. Start: npm start — then open http://localhost:3000

MCP Integration

Add to your Claude Desktop or Cursor MCP configuration:

{
  "mcpServers": {
    "calendar-consolidator": {
      "command": "node",
      "args": ["/path/to/calendar-consolidator-mcp/node/server/index.js"],
      "env": { "MCP_MODE": "true" }
    }
  }
}
